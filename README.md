# 💳 PocketPay

A modern fintech web application that allows users to securely manage digital payments.

## 🚀 Features

- User Signup
- User Login
- Dashboard
- Hide/Show Wallet Balance
- Responsive Design
- Form Validation
- Terms & Conditions
- Send Money (In Progress)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Git & GitHub
- Flask (still in progress)
